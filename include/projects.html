Project Title: Task Management App

Description: This project is a task management application built with Firebase for the backend. It allows users to add, update, delete, and track tasks. The app also includes features such as notifications, real-time task updates, and a user-friendly interface.

  • Built with HTML, CSS, JavaScript, and Firebase
  • Uses Firebase Authentication for secure user login
  • Real-time task updates with Firebase Realtime Database
  • Responsive design, ensuring usability across all devices
  • Includes a task notification system using Firebase Cloud Messaging
  • Features task filtering by status and priority
